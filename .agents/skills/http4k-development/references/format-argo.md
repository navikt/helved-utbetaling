# http4k-format-argo Reference

JSON node manipulation module backed by the Argo library. Implements `Json<JsonNode>` only — no auto-marshalling of typed objects.

## Construction

```kotlin
// Singleton — no configuration needed
val json = Argo
```

## JSON Node Manipulation

Argo uses `argo.jdom.JsonNode` as its node type.

```kotlin
// Parse and format
val node = Argo.parse("""{"name": "http4k", "version": 4}""")
Argo.compact(node)              // compact string
Argo.prettify(jsonString)       // prettify a JSON string
Argo.compactify(jsonString)     // compactify a JSON string

// Inspect nodes
Argo.typeOf(node)               // JsonType.Object
Argo.fields(node)               // [("name", JsonNode), ("version", JsonNode)]
Argo.elements(arrayNode)        // Iterable<JsonNode>
Argo.text(stringNode)           // String
Argo.bool(boolNode)             // Boolean
Argo.integer(numNode)           // Long
Argo.decimal(numNode)           // BigDecimal
Argo.textValueOf(node, "name")  // "http4k"

// Construct nodes
Argo {
    obj(
        "name" to string("http4k"),
        "tags" to array(listOf(string("kotlin"))),
        "count" to number(42),
        "active" to boolean(true),
        "extra" to nullNode()
    )
}

// Primitive conversions
"hello".asJsonValue()
42.asJsonValue()
true.asJsonValue()
```

## Lens Integration

```kotlin
// Raw JSON node body lens (no typed auto-marshalling)
val jsonLens = Body.json().toLens()
val node: JsonNode = jsonLens(request)
val response = Response(OK).with(jsonLens of node)

// JSON lens on any string spec
val queryJson = Query.string().json().required("data")
```

## Gotchas

- **No auto-marshalling**: Argo implements `Json<JsonNode>` only, not `AutoMarshallingJson`. There is no `Body.auto<T>()` — use `Body.json()` for raw node manipulation or pair with another format module for typed marshalling.
- **Number type detection**: Argo distinguishes integers from decimals by checking whether the number has a decimal part — `42` is `JsonType.Integer`, `42.0` is `JsonType.Number`.
- **Lightweight choice**: Argo is minimal — use it when you only need JSON node construction/inspection without object mapping.
- **Unstable hashcodes**: Object hashcodes generated by Argo may vary across versions. Do not rely on hashcode stability for caching or deduplication.
